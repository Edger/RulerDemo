# RulerDemo

通过使用自定义 View 来实现刻度尺。

- **RulerView** (com.android.ruler.widget.RulerView)

  通过滑动指针调节频道

- **RulerView2** (com.android.ruler.widget.RulerView2)

  通过滑动刻度调节频道

具体实现见代码，有详细注释。

![RulerView 和 RulerView2](https://github.com/Edger/RulerDemo/blob/master/ScreenRecord/Screenrecorder.gif)
